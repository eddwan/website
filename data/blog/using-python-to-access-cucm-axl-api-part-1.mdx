Hello Folks! O post de hoje iniciarei a apresentação sobre como usar o Python para acessar a API AXL do CUCM. Todos nós sabemos que o Python é uma linguagem de programação como uma curva de aprendizagem muito alta, o que faz dela a linguagem mais comum entre os engenheiros das mais diversas áreas para execução e automatização de tarefas. O propósito aqui não será apresentar os conceitos básicos da linguagem Python nem como orientar com relação aos passos para instalação e configuração da mesma em seu ambiente. Vamos começar! Para isso precisaremos das seguintes ferramentas:

- Python 3 instalado e funcionando
- AXLSQLToolkit que pode ser obtido através da página Plugins do CUCM
- Módulo [Zeep](https://pypi.org/project/zeep/) que pode ser instalado via PIP
- Módulo [Json2XML](https://pypi.org/project/json2xml/) que também pode ser instalado via PIP
- Um ambiente com CUCM instalado para executar os testes
- O [guia de referência](https://developer.cisco.com/docs/axl-schema-reference/) do AXL da página DevNET da Cisco
